# 简历模板

# WonderCV

**WonderCV (opens new window)**这个网站的厉害之处在于：

它是由几位非常厉害的人力资源专家创办的，也获得了很多世界500强HR的推荐。

目前功能也很强大：专业版本简历，支持中英文双模板，能够免费导出PDF版本，有专业的系统评测简历问题，有很多名企大牛的简历可以参考。

想要这些模板的，直接点 WonderCV (opens new window) 直达。

# Chuangkit

Chuangkit (opens new window) 是一款超级强大的在线设计工具，能像做PPT一样，通过拖动元素，就能做出好看的海报。

同样的，Chuangkit 也支持在线简历编辑，比较适合设计、创意领域的同学，简历模板的质感都是非常不错的。

# Canva可画

Canva (opens new window) 是一个超级强大的设计网站，来源于国外，不仅可以免费设计和下载简历，有各种模板可以选择。

Canva的在线编辑和修改功能也很强大，就像操作PPT一样，很方便，各种素材和字体也都是免费使用。

可以导出PDF、PNG等多种格式，打印也比较方便，关键是这个网站不仅有简历，还可以做海报、剪视频，简直是一个宝藏网站！

# 五百丁简历

网站：https://www.500d.me/ (opens new window)

汇聚了各个行业的简历，涉及的职业类别也非常宽泛，简历非常有设计感，强烈推荐！

模板数量也很多，可以说你想要什么类型的，这里都有~

# 职徒简历

网站：https://www.52cv.com/ (opens new window)

一个适用于金融、证券等领域的简历工具网站，非常多贴心的建议和大牛模板可以参考。

# 智能简历

网站：https://www.coderutil.com/jianli (opens new window)

CODER RESUME 是一个极简、易上手、针对程序员的一款在线简历创建平台，支持FORM表单、JSON两种创建模式，打造标准化的程序员简历平台。

免费，而且生成出来的格式很工整，支持PDF导出（1元去水印，也比较良心）。

# 一些有趣的开源简历模板

地址：https://github.com/dyweb/awesome-resume-for-chinese (opens new window)

链接里面是一些开源的简历模板，有的使用markdown构建，有的通过网页实现，小伙伴可以选择自己喜欢的风格。

如果小伙伴喜欢简约风，可直接选择markdown模板，生成自己的简历。

如果小伙伴喜欢个性，也可选择使用网页模板，自己修改样式，形成自己的风格，让自己的简历脱颖而出，还可以进行打包部署，让hr通过链接进行访问。
